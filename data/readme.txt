Due to GitHub's file size restrictions, the dataset used in this study has been archived into multiple split volumes. Please ensure that all volumes are downloaded to the same directory before decompression.

Download all split volumes into the same directory.

Ensure that you have 7-Zip or WinRAR installed.

Right-click on the first volume (e.g., files ending in .001 or .part1).

Select "Extract Here" or "Extract files..." to decompress the dataset.
